Tipos de campos

Tipo de campo Tag em HTML
Entrada de senhas input type="password"
Botão de limpar dados input type="reset"

Campos obrigatórios

Para tornar campos do formulário obrigatórios, inserir o atributo required dentro de cada input escolhido.

Definir tamanho do conteúdo

Para definir o tamanho mínimo, usar a tag minlength="".

Para definir o tamanho máximo, usar a tag maxlength="".

Definir tamanho da caixa

Para definir o tamanho da caixa (em caracteres), usar a tag size="".
