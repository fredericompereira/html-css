Tipos de campos

Tipo de campo Tag em HTML
E-mail input type="email"
Telefone input type="tel"

Expressões regulares

É possível definir padrões para valores inseridos nos campos usando expressões regulares (RegEx).

Para isso, usar a tag pattern="". Dentro da tag, usar ^ para iniciar uma expressão e $ para terminá-la.

Agrupamento de itens

Para agrupar itens dentro do formulário, usar a tag fieldset. Dentro dela, é possível criar uma legenda com a tag legend.
