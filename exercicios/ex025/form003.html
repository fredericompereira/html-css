Tipos de campos

Tipo de campo Tag em HTML
Números input type="number"
Mês e ano input type="month"
Dia, mês e ano input type="date"
Hora input type="time"

Atributos de valor mínimo e máximo de número

Os atributos min="" e max="" definem os valores mínimo e máximo para input type="number".

Atributo para intervalo de número

O atributo step="" define de quanto em quanto o número irá aumentar ou diminuir, como por exemplo de 0,5 para 1.

Atributo para campo já preenchido

O atributo value="" define um valor para o campo já ser preenchido.
