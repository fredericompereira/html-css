Primeiro formulário

Criando o básico

Tag para criar formulário: form.

Tag para criar campo de entrada de dados: input.

  • A propriedade name é mais importante para HTML e PHP.
  • A propriedade id é mais importante para JavaScript.

Tag para criar botão de enviar: input:submit.

Configurações

Para desligar a função de autocompletar, inserir a tag autocomplete="off" dentro de form.

Para enviar os dados de um formulário, usar a tag action dentro de form.

Uso de labels

Usar labels é importante porque ajuda os mecanismos de busca e navegadores a entenderem quais rótulos e campos são relacionados.

Para fazer a ligação entre um rótulo e um campo, usar a tag label. Depois, inserir o id do campo.

Métodos GET e POST

GET e POST são os métodos usados para enviar os dados de um formulários. Por padrão, o HTML usa o método GET.

O método GET exibe os dados enviados na URL do site.

O método POST esconde os dados da URL.

Quando usar GET ou POST?

GET POST
Os dados não forem sensíveis (nome, sobrenome, peso etc). Os dados forem sensíveis (senha, número do cartão, endereço etc), para enviar fotos ou outro tipo de arquivo.
Gerar URL compartilhável (igual o serviço de busca do Google, por exemplo). Se quiser que os dados não apareçam na URL do site.
Enviar dados de até 3 mil bytes (aproximadamente 3 mil letras). Se os dados tiverem mais de 3 mil bytes.

Obs: a diferença de velocidade entre os dois não é mais um critério considerável de performance.
