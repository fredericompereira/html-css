Tipos de campos

Tipo de campo Tag em HTML
Seletor de item select
Lista de dados (permite digitar) datalist
Caixa de mensagem textarea

Criar opções na lista

Dentro da tag select ou datalist, usar a tag option para criar itens.

É possível agrupar as opções de uma lista por meio da tag optgroup. Dentro dela, usar label para identificar o grupo.

Para deixar um item pré-selecionado, usar a tag selected.

Para exibir os itens de uma datalist, é necessário criar um input type="text" e usar a tag list="" para fazer referência ao id da lista.

Obs

Ao usar a tag textarea, não se deve usar o método get.
