Aula 8 - Checkbox e radio button

Teste de formulário

Tipos de campos

Tipo de campo Tag em HTML
Radio button (escolher só uma opção) input type="radio"
Checkbox (marcar mais de uma opção) input type="checkbox"

Obs: Dica para evitar um erro comum

Um erro comum ao usar radio é permitir que sejam selecionadas mais de uma opção. Para evitar isso, usar o mesmo name para todas as opções.

Definir uma opção como marcada

Dentro do input, usar a tag checked.
