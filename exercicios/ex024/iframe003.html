Aula 5 – Conteúdo no iframe por código

Navegando no iframe

Para abrir links dentro de um iframe, usar as seguintes propriedades:

  • Dentro do link - target="".
  • Dentro do iframe - name="".

Dê um nome ao frame e insira-o dentro de target.

Adicionar código dentro do iframe

Para inserir um conteúdo dentro do iframe por código HTML, usar o parâmetro srcdoc="".
