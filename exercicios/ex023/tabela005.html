Aula 13 – Agrupando colunas com colgroup

Exercício de tabela

Para agrupar colunas, utilizar a tag colgroup.

Dentro dela, criar tags col para as colunas da tabela.

Para personalizar uma coluna, criar um id ou class e usar na folha de estilos.

Também é possível criar uma col para várias colunas usando a tag span.
