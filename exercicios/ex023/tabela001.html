Primeira tabela

Tag usada para criar tabela: table.

Hierarquia de tabelas simples

  • Table = Tabela
    • Table row = Linha de tabela
      • Table header = cabeçalho de tabela
      • Table data = dado de tabela

Propriedade border-collapse permite separar ou unir as bordar em uma só.

  • Valor collapse une as bordas.
  • Valor separate separa as bordas.

Obs: é opicional o fechamento das tags tr e td.


Propriedade verical-align permite alinhar verticalmente os dados da tabela.

Exemplo:

A1 B1 C1
A2 B2 C2
A3 B3 C3
A4 B4 C4
