Aula 6 – Inconvenientes do iframe

Inconvenientes do iframe

1. Não confundir iframe com frame

Frames não são suportados em HTML5. Eles eram usados em HTML4 para definir uma janela específica.

Iframes são utilizados para muitos recursos úteis, como incorporar vídeos do YouTube por exemplo.

2. Problemas com mecanismos de busca

Podem haver problemas de indexação com o uso de iframes no site. O robô do Google talvez tenha dificuldade de acessar o conteúdo do iframe.

Dica: não utilizar muitos iframes no site.

3. Problemas de acessibilidade

Leitores de tela apresentam problemas ao acessar iframes.

4. Problemas de usabilidade

O navegador ou o usuário podem acabar se confundindo com iframes no site.

Por exemplo: ao navegar entre páginas dentro de um iframe, o navegador pode se confundir em qual página voltar.

5. Problemas de segurança

Ao inserir um site de outra pessoa dentro de um iframe, é preciso tomar cuidado para não estar colocando uma página vulnerável a ataques dentro do seu site.
